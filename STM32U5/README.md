# STM32U5 Gesture Recognition

This directory is reserved for the STM32U5 implementation of the gesture-recognition project.

Planned contents:

- STM32U5 model-training notes.
- TensorFlow Lite Micro or other deployment artifacts.
- Quantized model files or generated C arrays when appropriate.
- Firmware source and build instructions.
- Measurement results for latency, memory, and accuracy.
- Host-side demo instructions if the STM32U5 demo uses a serial or camera viewer.

The MAX78000 implementation is available in `../MAX78000/` and can be used as a reference for the expected documentation structure.
