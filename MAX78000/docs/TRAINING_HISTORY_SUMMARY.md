## File Coverage
The project includes these reusable source/config groups:

- Canonical ai8x model Python files: `training/ai8x_custom_files/ai8x_training_models/`
- Canonical ai8x dataset adapter Python files: `training/ai8x_custom_files/ai8x_training_datasets/`
- Canonical QAT policy YAML files: `training/ai8x_custom_files/qat_policies/`
- ai8x-synthesis network YAML files: `training/ai8x_custom_files/synthesis_networks/`
- Original upload bundles used on the server: `training/ai8x_custom_files/codex_uploads/`
- Selected deployable model metadata and labels: `models/best_deployable/`
- Full generated deployed weights for the stable firmware: `mcu/firmware/stable/weights.h`

Large PyTorch checkpoints remain excluded from this source tree; see `docs/MODEL_ARTIFACTS.md` for the artifact policy.
